# XR 手势追踪 UI 交互系统
**贡献者：kemomi**

## 核心设计：仿 SAO 呼出菜单
“食指与中指并拢，由向上划变为向正下方挥动”即打开主菜单。
- 菜单呈半透明环形悬浮在左手腕上方或正前方。
- 使用射线或直接手指点选。

## 主菜单环形结构
- **12点钟方向**：角色状态（HP条、属性、装备栏）
- **2点**：物品（环形背包，分类页签）
- **4点**：技能（已学会剑技树，可拖拽至快捷栏）
- **6点**：地图（当前楼层小地图，支持捏合缩放）
- **8点**：系统/设置
- **10点**：社交/好友

## 手势交互规范
| 操作 | 手势 | 说明 |
|------|------|------|
| 呼出/关闭菜单 | 双指下划（食指+中指） | 无需按住，做一次动作即可 |
| 点选 | 伸出食指，指尖触碰按钮 | 触碰后有轻微触觉反馈 |
| 拖动/滚动列表 | 捏合（拇指食指捏合）并拖动 | 模拟拖拽纸条感觉 |
| 缩放地图 | 双手捏合并拉开/合拢 | 与MR手势一致 |
| 返回上层 | 手掌由外向内挥动 | 或左手轻拍右手手背 |

## UI 动画效果
- 菜单从呼出位置径向扩散展开，带有淡入和弹性动画。
- 选中按钮会浮起并散发粒子光点（类似蓝白色宝石光辉）。
- 切换菜单分类时，环形菜单旋转并伴有轻微音效（取自 SAO UI 音效风格）。

## 适用设备兼容性
- **Meta Quest 3**：主要使用手柄射线交互，手势追踪为辅。
- **Apple Vision Pro**：完整手势支持，眼动追踪可辅助高亮注视的按钮，捏合确认为主操作。
